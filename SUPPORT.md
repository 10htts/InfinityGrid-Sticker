# Support

## Getting Help
- Search existing issues first.
- If your question is usage-related, open a discussion/issue with:
  - what you tried
  - expected behavior
  - actual behavior
  - screenshots/logs when relevant

## Bug Reports
Please include:
- OS and browser
- app version/commit
- reproduction steps
- sample input (tag config / icon / text) if relevant

## Security Issues
For security concerns, follow `SECURITY.md` and report privately.
